== Description ==
<pre>Only and first Visual Composer Addon of Tabs</pre>
Tabs for Visual Composer is really simple WordPress plugin and <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed">Visual Composer</a> addon that will allow user to put any content in tabs of different style and with different color scheming.You can also display content using shortcode.It contains really simple settings and easy to use.You can set it up in few seconds.You can display Posts,Pages,Portfolios,Testimonials,Gallery and many more.

<a href="http://thecodude.com/demo/codecanyon/advance-tabs/">Demo</a>

<h3>Features</h3>

<ul>
	<li>8 different styles</li> 
	<li>Predefined Color Scehemes </li> 
	<li>Horizontal Tabs</li> 
	<li>Vetical Tabs</li>  
	<li>Animated</li>
	<li>24/7 Support</li>
</ul>
